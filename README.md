# 🏠 AirBnB Clone

A full stack vacation rental web application inspired by Airbnb — built with React on the frontend and Node.js + Express + MongoDB on the backend.

---

## ✨ Features

- Browse and search rental listings
- View property details and photos
- User authentication (register / login)
- Responsive design for mobile and desktop

---

## 🛠️ Tech Stack

**Frontend**
- React
- HTML5 / CSS3
- JavaScript (ES6+)

**Backend**
- Node.js
- Express
- MongoDB

---

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB running locally (or connection string)

### Installation

```bash
# Clone the repository
git clone https://github.com/Yakirasor12/FullStack-AirBnB-clone.git
cd FullStack-AirBnB-clone
```

**Start the backend (API):**
```bash
cd api
npm install
npm start
```

**Start the frontend (in a new terminal):**
```bash
cd client
npm install
npm run dev
```

The app will be running at `http://localhost:5173`
